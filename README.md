# 功能
- 保留最後標籤頁（防止關閉最後一個標籤頁時關閉瀏覽器，點X不行）
- 滑鼠懸停標籤欄滾動
- 按住右鍵時滾輪滾動標籤欄
- 便攜設計，程序放在App目錄，資料放在Data目錄（不相容原版資料，可以重裝系統換電腦不丟資料）
- 移除更新錯誤警告（因為是綠色版沒有自動更新功能）
